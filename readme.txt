WOO Related Products Slider display woocommerce products in a crousel slider. The Slider will display on bottom area of single product page.

== Description ==

Demo: http://clownfishweb.com/demo/product/woo-related-products-slider/


= WOO Related Products Slider =

- Nice sleek slideshow.
- Touch Enabled Slider.
- 100% Responsive Slider
- Nice sleek design.
- Display Related products under single product page.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/PLUGIN_NAME' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's it! Now you can look at product single page. It's done.
